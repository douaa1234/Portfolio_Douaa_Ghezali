# Douaa Ghezali Portfolio

Static portfolio site built with `HTML`, `CSS` and `JavaScript`.

## Open locally

1. Open `index.html` directly in a browser.
2. Or start a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://127.0.0.1:8000`.

## Files

- `index.html` contains the content and page structure.
- `styles.css` contains the design system, layout, responsive styling and theme styles.
- `script.js` handles theme switching, reveal animations and active navigation highlighting.
- `DouaaGhezali_GraduateCV.pdf` is the downloadable CV linked from the site.
